v2.5.0 Latest Release

Released on October 24, 2023

auto_awesome New Features

  • Multi-Planar Reconstruction (MPR)

    Introduced full MPR support allowing radiologists to reconstruct images in coronal, sagittal, and oblique planes from the original axial data. Includes real-time adjustment controls.

    View Screenshot

  • Cobb Angle Measurement Tool

    Added a dedicated tool for orthopedic measurements to calculate spinal curvature directly within the viewer interface.

trending_up Improvements

  • Rendering Engine: Rendering performance for large CT datasets (500+ slices) has been optimized, resulting in a 30% reduction in initial load time.

  • UI Polish: Updated the contrast of measurement annotations to be more visible on bright backgrounds (e.g., Lung window).

bug_report Bug Fixes

  • Resolved an issue where window/level presets were not persisting after a page refresh for guest users.
